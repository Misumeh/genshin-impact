Frequently Asked Questions

Are these packs compatible with the latest Minecraft version?

Yes! While the base versions are tailored for 1.8 PvP, we actively update our premium packs to the latest Java versions (e.g., 1.20+), complete with custom GUI and items.

How do I download the Ad-Free or NSFW versions?

Ad-free downloads, latest version updates, and exclusive NSFW packs are provided as perks for supporting the project on Patreon. Free versions remain fully playable via ad-supported links.
